# Gherkin

## Description

GherkinはE2E (エンドツーエンド) テストでよく使われる記法。

実装のサンプルを作成する。

題材は「オンラインショップ」をテーマに実装してみます。

## Stack

作成時点で最新のバージョンで実装する

Frontend
- Remix.js
- Cucumber
- Playwright
- TypeScript

Backend
- Elixir/Phoenix
- PostgreSQL
